# 📈 Real-Time Stock Market Dashboard

A **professional real-time stock market dashboard** built with **Python, Streamlit, Plotly, and Yahoo Finance API** that allows users to analyze stock prices, visualize trends, and monitor live market movements with advanced technical indicators.

The dashboard automatically refreshes every **60 seconds** to provide updated stock market data.

---

# 🚀 Features

✅ Real-time stock price tracking
✅ Interactive **candlestick and line charts**
✅ Live market watchlist in sidebar
✅ Technical indicators for analysis
✅ Auto-refresh every **60 seconds**
✅ Volume chart visualization
✅ Historical stock data tables
✅ Clean professional Streamlit dashboard UI

---

# 📊 Technical Indicators Included

This dashboard includes several popular technical indicators used by traders:

| Indicator                            | Purpose                            |
| ------------------------------------ | ---------------------------------- |
| **SMA (Simple Moving Average)**      | Identifies long-term trends        |
| **EMA (Exponential Moving Average)** | Tracks recent price changes        |
| **RSI (Relative Strength Index)**    | Detects overbought/oversold levels |
| **MACD**                             | Shows momentum and trend reversals |
| **Bollinger Bands**                  | Measures market volatility         |

# ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/real-time-stock-dashboard.git
cd real-time-stock-dashboard
```

---

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Dashboard

Start the Streamlit application:

```bash
streamlit run stocks_dashboard.py
```

Open in browser:

```
http://localhost:8501
```

# 📊 Dashboard Components

## 1️⃣ Stock Selection

Users can enter any stock ticker symbol:

* AAPL
* TSLA
* MSFT
* AMZN
* GOOGL

---

## 2️⃣ Price Chart

Interactive charts include:

* Candlestick chart
* Line chart
* Bollinger Bands overlay
* Moving averages

---

## 3️⃣ Volume Analysis

A separate **volume chart** shows trading activity for the selected stock.

---

## 4️⃣ Technical Indicator Charts

Optional indicator charts include:

### RSI Chart

Shows momentum and identifies overbought or oversold conditions.

### MACD Chart

Displays trend strength and momentum shifts.

---

## 5️⃣ Live Market Watchlist

The sidebar displays real-time prices for major stocks:

* Apple (AAPL)
* Google (GOOGL)
* Amazon (AMZN)
* Microsoft (MSFT)
* Tesla (TSLA)

Prices automatically refresh every **60 seconds**.
